# Breast Cancer Classification 

## Description
Ce projet est une application de Machine Learning permettant de classifier les tumeurs du sein en **malignes** ou **bénignes** à partir de données médicales.  
Il utilise plusieurs algorithmes pour analyser les caractéristiques cellulaires et aider à la prise de décision.

## Dataset
Le projet utilise le dataset **Breast Cancer Wisconsin** provenant de Kaggle.  
Il contient **569 observations** et **30 variables numériques** représentant des mesures de cellules.  
La variable cible est **diagnosis** :
- **M** : Malin  
- **B** : Bénin  

## Objectif
L’objectif de ce projet est de construire un modèle capable de prédire si une tumeur est **maligne** ou **bénigne** à partir des données disponibles.
